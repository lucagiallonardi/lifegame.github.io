Bienvenidos al Juego de la Vida

Imaginemos que cada pixel es una celula...

Este Juego simula un sistema que evoluciona siguiendo las siguientes reglas:

  • Si una celula tiene menos de 2 celulas vecinas muere
  • Si una celula tiene 2 celulas vecinas permanece igual
  • Si una celula tiene 3 celulas vecinas se reproduce
  • Si una celula tiene mas de 3 celulas vecinas se muere
